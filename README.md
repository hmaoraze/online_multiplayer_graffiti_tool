# 涂鸦签名工具 (Doodle Sign)

一款面向毕业、聚会、纪念活动等场景的在线多人协作涂鸦签名工具，核心目标是还原"像在校服或纪念册上用记号笔自由签名"的真实体验。

## 功能特性

### 核心功能
- 🎨 **多种画笔工具**：马克笔、荧光笔、喷漆、橡皮擦
- 🎯 **颜色选择**：预设调色板 + 自定义颜色
- 📏 **大小调节**：根据不同工具自动调整可调节范围
- ↩️ **撤销/重做**：完整的操作历史记录
- 🗑️ **清空画布**：一键重置画布

### 协作功能
- 👥 **多人实时协作**：基于 Yjs 的实时同步
- 👤 **参与者列表**：显示当前在线用户
- 🖱️ **光标追踪**：实时显示其他用户的光标位置
- 🔗 **房间分享**：一键生成分享链接

### 导出功能
- 📸 **PNG 导出**：高清图片导出
- 📄 **PDF 导出**：适合打印的文档格式
- 🖼️ **SVG 导出**：矢量图形导出

### 用户体验
- 📱 **移动端优化**：支持触摸和触控笔
- 🎬 **流畅动画**：基于 Framer Motion 的交互动画
- 🌈 **温馨设计**：温暖的色彩方案和仪式感UI

## 技术栈

- **前端框架**: React 18 + TypeScript
- **构建工具**: Vite 5
- **样式方案**: TailwindCSS 3
- **状态管理**: Zustand 4
- **Canvas 渲染**: Konva + React-Konva
- **实时协作**: Yjs + y-websocket
- **动画效果**: Framer Motion 11
- **导出功能**: html2canvas + jsPDF
- **图标库**: Lucide React

## 使用说明

### 创建房间
1. 访问首页
2. 选择背景模板（毕业、校服、婚礼等）
3. 输入房间名称
4. 点击"创建房间"
5. 分享链接给朋友

### 加入房间
1. 输入房间ID或通过分享链接进入
2. 输入昵称（可匿名）
3. 开始在画布上书写

### 导出画布
1. 点击右上角的"导出"按钮
2. 选择导出格式（PNG/PDF/SVG）
3. 下载到本地

## 实时协作

应用使用 Yjs 和 WebSocket 实现实时协作功能。目前配置使用公共的 Yjs demo 服务器，生产环境建议部署自己的 y-websocket 服务器。

### 协作架构
- **Y.Doc**: 共享文档，包含笔迹、贴纸、用户信息
- **Awareness**: 用户状态同步（光标位置、在线状态）
- **WebsocketProvider**: WebSocket 连接管理

## 性能优化
